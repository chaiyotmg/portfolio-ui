# Portfolio UI (Terminal Aesthetic)

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: [TailwindCSS](https://tailwindcss.com/)
- **Runtime**: [Bun](https://bun.sh/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/) & Custom SVG Assets
- **Type Safety**: TypeScript

## 🚀 Getting Started

Ensure you have [Bun](https://bun.sh/) installed on your system.

### Installation

```bash
bun install
```

### Development

```bash
# Start the development server
bun run dev
```

### Build

```bash
# Create a production build
bun run build
```

## 📂 Project Structure

```text
src/
├── app/            # Next.js App Router pages
├── components/     # Atomic Design structure (Atoms, Molecules, Organisms)
├── data/           # Portfolio data (portfolioData.ts)
├── domain/         # TypeScript entities and interfaces
```
